Catalog Server

A catalog server is a server that serves a catalog of products.

Routes

GET /products

Returns a list of all products in the database.

POST /products

Adds a new product to the database.

GET /products/:id

Returns a single product from the database.

PUT /products/:id

Updates a product in the database.

DELETE /products/:id

Deletes a product from the database.

POST /auth/login

Logs in a user.

POST /auth/register

Registers a new user.

POST /search

Searches for a product in the database.
